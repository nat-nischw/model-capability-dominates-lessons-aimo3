# data/

## reference.csv

10 AIME-3 reference problems (`id`, `problem`, `answer`) used by
[`utils/optimize_prompt.py`](../utils/optimize_prompt.py) to swap 5 hard
problems into the GEPA test set and 5 easier problems into the validation set.

**Source:** AI Mathematical Olympiad - Progress Prize 3 (Kaggle) —
<https://www.kaggle.com/competitions/ai-mathematical-olympiad-progress-prize-3/data?select=reference.csv>
